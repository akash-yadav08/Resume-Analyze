# 🚀 AI Resume Analyzer

An AI-powered Resume Analyzer that evaluates resumes, calculates a skill-based score, and provides improvement suggestions through an interactive UI.

---

## 📌 Features

- 📄 Upload Resume (PDF)
- 📊 Resume Score (visual progress bar)
- 🏷️ Skill Detection (auto extraction)
- ❌ Missing Skills Identification
- 💡 Suggestions for improvement
- 🧠 AI-based Summary (optional integration)
- 🌐 Interactive Frontend UI

---

## 🛠️ Tech Stack

### Backend
- Python (FastAPI)
- REST API
- PDF Processing (pdfminer)

### Frontend
- HTML
- CSS (Modern UI)
- JavaScript

### DevOps
